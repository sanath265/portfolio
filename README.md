# portfolio

## Getting Started

To clone the project, run:
```bash
git clone https://github.com/your-username/portfolio.git
cd portfolio
```

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm start
```

Build for production:
```bash
npm run build
```
